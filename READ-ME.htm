Description:

This modification allows users to select from a variaty of skins which one they would like to use on the forum. Each Member is able to use a different skin.

This Modification works for both registed members and guests.

You can use to store the chosen skin in either a cookie or in the database.
(Note: the database option only works for registered members, and guests will continue to use cookies to store their selected skin.)

You can new skins by editing the 's2h_skinfile.asp' found in the folder 'includes'. There is a tutorial on how to do this here
